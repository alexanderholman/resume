Current Research

My current doctoral research investigates whether machine-learned interatomic potentials, particularly MACE, can reproduce density functional theory derived rankings of interface favourability across semiconductor and heterostructure systems. The work sits within computational materials physics and draws on condensed matter, crystallography, applied quantum mechanics, and quantum chemistry. The core question is not simply whether an ML potential matches absolute DFT energies, but whether it preserves the ordering well enough to become a practical screening tool for large interface search spaces.

This work focuses on atomistic interfaces generated across a range of group IV systems including carbon, silicon, germanium, and tin. I study how interface energetics, lattice matching, surface orientation, structural relaxation, and chemical composition affect the agreement between machine learning models and first-principles methods.

A major theme in the project is methodological honesty. In practice, interface prediction is complicated by relaxation effects, structural sensitivity, and the combinatorial growth of possible configurations. My research therefore combines direct comparison, careful error analysis, and workflow design, rather than treating machine learning as a black-box replacement for DFT.

Research Themes

The main themes that run through my current work are:

1. Interface stability and structure prediction

Understanding which atomic arrangements at a material interface are energetically favourable, and how those arrangements depend on orientation, registry, strain, and composition.

2. Machine-learned interatomic potentials

Evaluating when a potential such as MACE is reliable for ranking interface candidates, where it fails, and how it may be improved through retraining, delta-learning, or hybrid workflows.

3. DFT versus ML workflow design

Using DFT where accuracy is essential, while exploiting ML models for scale, screening, and fast iteration in large configuration spaces.

4. Automated interface generation

Developing and using algorithmic pipelines for constructing realistic interfaces through lattice matching, surface selection, registry shifts, rotations, supercell scaling, and defect-aware modelling.

5. Scientific software and compute systems

Building the scripts, data pipelines, and computational infrastructure needed to run large atomistic campaigns reproducibly across local and HPC environments.

PhD Project

My PhD centres on the prediction of interface favourability in solid-state systems. Interfaces often determine the behaviour of real devices, but they are difficult to model well because the relevant configuration space grows rapidly with surface choice, relative shift, composition, and defect structure. Density functional theory provides an accurate route to interface energetics, but is too computationally expensive to use naively at large scale.

To address this, I investigate whether machine-learned interatomic potentials can preserve DFT-based rankings across many candidate interfaces. This includes generating interface structures, relaxing them under different methods, comparing formation-energy trends, and analysing rank correlation through metrics such as Spearman rho, Kendall tau, top-N overlap, RMSE, and MAE.

The broader aim is to contribute to a workflow in which machine learning is used not as a replacement for physics, but as a carefully tested accelerator for interface discovery. Longer term, this points toward a more intelligent generative model for interface discovery rather than brute-force combinatorial stacking. This has relevance to semiconductor heterostructures, grain boundaries, and other systems where interface structure strongly influences behaviour.

Methods and Technical Stack

My work draws on a mixture of physics, numerical simulation, machine learning, and software engineering. The main tools and methods I use include:

  • Density functional theory, especially VASP-based workflows for interface relaxation and energy evaluation
  • Machine-learned interatomic potentials, especially MACE, for atomistic prediction and comparative evaluation
  • Interface generation workflows based on crystallographic matching, Miller planes, supercell construction, strain minimisation, and interface registry analysis
  • Python-based scientific tooling for data preparation, automation, analysis, plotting, and workflow control
  • High performance computing environments for large batch calculations and method comparison
  • Statistical comparison methods for ranking quality, including rank correlation and top-N analysis
  • Reproducible data pipelines for managing thousands of atomistic structures, configuration-specific inputs, and interface-screening workflows

Selected Research Contributions

Across my doctoral work so far, I have been building both the scientific case and the computational machinery needed for large-scale interface studies. Key contributions include:

  • Comparative analysis of DFT and machine-learned interatomic potential rankings for large sets of interface structures
  • Detailed study of how structural relaxation influences disagreement between DFT and ML models
  • Interface generation workflows spanning binary and broader group IV systems
  • Use of rank-preservation metrics to evaluate practical screening value rather than relying only on absolute error
  • Development of computational pipelines for handling large numbers of structure-specific simulation inputs
  • Ongoing work toward improved hybrid strategies such as delta-learning and feature-based interface prediction
